# Network-Security-Design-and-Implementation
This project focuses on designing and implementing a secure network infrastructure using Cisco Packet Tracer, Nessus, and OpenVPN. It involves creating a detailed network topology, configuring firewalls and access control policies, and performing vulnerability assessments to identify and mitigate security risks. The project also includes setting up a secure VPN connection to enable encrypted remote communication. By integrating multiple security layers, it demonstrates key network security principles such as defense-in-depth, risk management, and secure data transmission. The overall objective is to provide a practical understanding of modern network security practices and tools used to protect organizational systems and data.

The network topology was designed using Cisco Packet Tracer and consists of multiple components including routers, switches, servers, and a remote user. The network is divided into three main segments:
•	Internal LAN (Local Area Network)
•	DMZ (Demilitarized Zone)
•	External Network (Remote user)
Two routers were used to simulate real-world network architecture. One router acts as a gateway to the external network, while the other functions as a firewall controlling traffic between different network segments.
